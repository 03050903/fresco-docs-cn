你可以从Maven库下载:

## Gradle:

```groovy
dependencies {
  compile 'com.facebook.fresco:fresco:0.1.0+'
}
```

## Maven:

```xml
<dependency>
  <groupId>com.facebook.fresco</groupId>
  <artifactId>fresco</artifactId>
  <version>LATEST</version>
</dependency>
```
