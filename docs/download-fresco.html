下载Fresco 编辑和纠错

类库发布到了Maven中央库:

Gradle:

dependencies {
  compile 'com.facebook.fresco:fresco:0.2.0+'
}

Maven:

<dependency>
  <groupId>com.facebook.fresco</groupId>
  <artifactId>fresco</artifactId>
  <version>LATEST</version>
</dependency>
